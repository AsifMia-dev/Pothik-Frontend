# Pothik-Frontend

**Pothik-Frontend** is a modern frontend application built using **React** and **Vite**.  
It uses **PrimeReact** for UI components, **PrimeIcons** for icons, **React Router** for routing, and **PrimeFlex** for responsive layouts.  

This project is part of the Pothik platform and is designed for building interactive and scalable web applications.

## Prerequisites

Before running the project locally, make sure you have the following installed:

- [Node.js](https://nodejs.org/) >= 18.x
- npm >= 9.x (comes with Node.js) or yarn
- Git

---

## Project Setup on Local Machine

Follow these steps carefully to run the project on your local machine:

### 1. Clone the repository

Open your terminal or command prompt and run:

```bash
git clone https://github.com/AsifMia-dev/Pothik-Frontend.git
```
### 2. Navigate into the project folder
```bash
cd Pothik-Frontend
```

### 3. Install project dependencies
```bash
npm install

npm install react react-dom

npm install tailwindcss @tailwindcss/vite

npm install primereact

npm install primeicons

npm install primeflex

npm install axios
```

### 4. Run the project
```bash
npm run dev
